# Z.A.R.A. (Zerkg's Audiosurf Rich Application).
Small rich app, that helps you in stats analyze (currently have 2 modules: Leaderboard Analyzer and Score Analyzer).
To use modules, firts of all, you need to find a leaderboard, use leaderboard searcher component: just type artist or song name (also you can paste the leaderboard URL directly).
When ZARA found the leaderboard, you need to select a score. Once the score selected, the module will analyze it and display to use the analyze.
